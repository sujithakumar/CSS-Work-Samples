Combinators

Descendant Selector (space) matches all elements that are descendants of a specified element. div span{ border: 2px solid yellowgreen; }
child selector (>) selects all elements that are children of a specified element. p>span{ color:blue; }
adjacent sibling selector (+) Selects element that is the adjacent/first sibling of a specified element. Sibling elements must have the same parent element, and "adjacent" means "immediately following". strong+section{ background-color: aquamarine; }
general sibling selector (~) Selects all the elements that are siblings of the specified elements. span ~ span{ background-color: brown; }
