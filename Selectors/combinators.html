Combinators

Descendant Selector (space) matches all elements that are descendants of a specified element. div p { color: hotpink; }
child selector (>) selects all elements that are direct children of a specified element. div>span { border: 2px solid yellow; }
adjacent sibling selector (+) Selects element that is the adjacent/first sibling of a specified element. Sibling elements must have the same parent element, and "adjacent" means "immediately following". div+span{ background-color: violet; } Span, that immediately follows div will be selected
general sibling selector (~) Selects all the elements that are siblings of the specified elements. div~p { background-color: palegreen; } all P element that are siblings of div
