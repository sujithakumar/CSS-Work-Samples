Universal Selector : *

Applies styles on every element in the document

example : font style of this document

Element Selector : tag name

Applies styles on elements in the document

example : paragraph styles- info inside every details tag are made of P tag

id Selector : #

Applies styles on element with id="idSelector" in the document

id is set in 2 & 4th summary tag

class Selector : .

Applies styles on every element with class ="div-borders" in the document

note: styles are inherited from external css imported here
